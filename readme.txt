== Description ==

Flarum Bridge allows WordPress users to sign-in to a sub-directory Flarum installation that exists on the same domain as your WordPress site.

== Installation ==

= Minimum plugin requirements =

* PHP 5.6
* WordPress 4.7

= Manual installation =

Download the Flarum Bridge plugin and upload it to your server. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Frequently Asked Questions ==

= Where do I get the Flarum API key from? =

You will need to generate and add this manually to the Flarum api_keys database table.
